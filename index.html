ECWolf

Game file source

Notes
  • Uses ecwolf-js.
  • "Game files" here refers to IWADs, e.g. .wl6 or .sod files.
  • ECWolf officially supports Wolfenstein 3D, Spear of Destiny, and Super 3D Noah's Ark.
  • The recommended way of playing the game is in a fullscreen Chrome window. This way the browser will not interrupt the game with its hotkeys (e.g. Ctrl+W). Unfortunately, Firefox does not (yet) support the Keyboard Lock API.
